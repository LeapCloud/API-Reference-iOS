Overview

A MaxLeap Framework Passport Object that is a local representation of a passport persisted to the MaxLeap. This class is a subclass of a MLObject, and retains the same functionality of a MLObject, but also extends it with various passport specific methods, like authentication, signing up, and validation uniqueness.
