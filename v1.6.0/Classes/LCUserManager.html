LCUserManager Class Reference

Inherits from NSObject
Declared in LCUserManager.h

Overview

The helper class of LCUser. This class extends the behaviour of LCObject, like signing up, login, and validation uniqueness.

Other Methods

+ enableAutomaticUser

Enables automatic creation of anonymous users. After calling this method, [LCUser currentUser] will always have a value. The user will only be created on the server once the user has been saved, or once an object with a relation to that user or an ACL that refers to the user has been saved.

+ (void)enableAutomaticUser

Discussion

Note: saveEventually will not work if an item being saved has a relation to an automatic user that has never been saved.
