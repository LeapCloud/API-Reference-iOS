Inherits from NSObject

Overview

The LCPush class defines a push notification that can be sent from a client device.

The preferred way of modifying or retrieving channel subscriptions is to use the LCInstallation class, instead of the class methods in LCPush.

This class is currently for iOS only. LeapCloud does not handle Push Notifications to LeapCloud applications running on OS X. Push Notifications can be sent from OS X applications via Cloud Code or the REST API to push-enabled devices (e.g. iOS or Android).
