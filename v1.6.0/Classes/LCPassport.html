Overview

A LeapCloud Framework Passport Object that is a local representation of a passport persisted to the LeapCloud. This class is a subclass of a LCObject, and retains the same functionality of a LCObject, but also extends it with various passport specific methods, like authentication, signing up, and validation uniqueness.
