LCConfigManager Class Reference

Overview

LCConfigManager helps you retrieve config and observe the current config value changes.

Current Config

+ currentConfig

Returns the most recently fetched config.

+ (LCConfig *)currentConfig

Return Value

Current, last fetched instance of LCConfig.

Discussion

If there was no config fetched - this method will return an empty instance of LCConfig.

Observe Config Changes

+ addObserver:forKey:valueChangedHandler:

Registers anObserver to receive config value changed notifications for the specified key relative to the current config. The observer is not retained.

+ (void)addObserver:(NSObject *)observer forKey:(NSString *)key valueChangedHandler:(LCConfigValueChangedBlock)block

Parameters

observer

The object to register for config value changed notifications.
key

The key, relative to the current config, of the property to observe. This value must not be nil.
block

The callback to be excuted when value for the specified key changed. It should have the following argument signature: ^(id newValue, id oldValue).
