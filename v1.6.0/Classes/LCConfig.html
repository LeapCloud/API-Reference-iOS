LCConfig Class Reference

Inherits from NSObject
Declared in LCConfig.h

Overview

LCConfig is a representation of the remote configuration object. It enables you to add things like feature gating, a/b testing or simple “Message of the day”.

Parameters

– objectForKey:

Returns the object associated with a given key.

- (id)objectForKey:(NSString *)key

Parameters

key

The key for which to return the corresponding configuration value.

Return Value

The value associated with key, or nil if there is no such value.

Declared In

LCConfig.h

– objectForKeyedSubscript:

Returns the object associated with a given key.

- (id)objectForKeyedSubscript:(NSString *)keyedSubscript

Parameters

keyedSubscript

The keyed subscript for which to return the corresponding configuration value.

Return Value

The value associated with key, or defaultValue if there is no such value.

Discussion

This method enables usage of literal syntax on LCConfig. E.g. NSString *value = config[@"key"];
