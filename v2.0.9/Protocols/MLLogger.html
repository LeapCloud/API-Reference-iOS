Overview

The logger interface.

– setLogLevel:

Sets the level of logging to display.

- (void)setLogLevel:(MLLogLevel)level

Parameters

level

Log level to set

Discussion

By default: - If running inside an app that was downloaded from iOS App Store, it is set to MLLogLevelNone. - All other cases, it is set to MLLogLevelWarning. - When [MLLogger setLogLevel:] get called, it will attempt to call this method on current logger.
