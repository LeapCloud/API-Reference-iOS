+ queryWithQueryStringElement:

Create SearchQuery using a query_string element.

+ (instancetype)queryWithQueryStringElement:(NSDictionary *)queryStringElement

Parameters

queryStringElement

the query_string element

Return Value

A search query instance.

Discussion

The structure of search query:

{
    "query":{
        "query_string":queryStringElement
    },
    "from":0,
    "size":10
}

Sample for queryStringElement:

@{
    @"query" : @"text to search"
 }

For more infomation about query_string, please refer to Query DSL - Query String Query
